REST Service (.NET Core)

The REST SErvice (.NET Core) project is a Visual Studio Integration Extension for creating REST Services. This extension will run Visual Studio versions 2019 and 2022. The extension offers the following features:

  • OAuth2/Open-Id Connect support
  • >.NET 6.0
  • Swagger
  • Integrated Database support - SQL Server, Postgresql or MySQL
  • Correlation Id support
  • Repository and Orchestration layers support
    • An integrated RQL engine
